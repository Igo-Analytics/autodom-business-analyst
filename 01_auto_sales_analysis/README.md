# Анализ эффективности дилерской сети

## Цель проекта
Создание системы мониторинга KPI для 35+ дилерских центров Автодома.

## Результаты
- Автоматизация ежедневной отчетности (сокращение времени на 94%)
- Выявление лучших практик среди ДЦ
- Создание единой системы метрик

## Файлы
- `auto_sales_analyzer.py` - основной скрипт анализа
- `dealer_kpi_analysis.sql` - SQL запросы для анализа
